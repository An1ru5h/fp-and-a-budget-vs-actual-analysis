# 📊 Corporate FP&A Budget vs Actual Analysis Dashboard

## Overview

This project is an end-to-end **Financial Planning & Analysis (FP&A)** case study that compares **budgeted** and **actual** financial performance. The objective is to help management monitor financial performance, identify budget variances, evaluate departmental spending, and support data-driven decision-making through interactive dashboards and financial analysis.

The project demonstrates a complete analytics workflow, including data cleaning, exploratory data analysis (EDA), KPI development, variance analysis, financial reporting, and dashboard creation.

---

## Business Problem

Organizations prepare budgets to plan their financial performance. However, actual spending and revenue often differ from planned budgets. Understanding these differences is essential for improving financial planning, controlling costs, and making informed business decisions.

This project answers questions such as:

- How does actual performance compare to the budget?
- Which departments exceeded or stayed within budget?
- Which categories contribute most to budget variance?
- What are the monthly financial performance trends?
- What recommendations can improve future budgeting?
